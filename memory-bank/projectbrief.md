# Project Brief

## Project
Webflow UI Mapper

## Purpose
Transform a target website into Webflow collections, symbols, and bindings using AI-driven analysis and export.

## Primary Goals
- Accept a URL and generate Webflow-ready JSON/CSV outputs.
- Provide authenticated access (email/password and Google OAuth).
- Protect analysis features behind authentication.

## Core Scope
- Web scraping with Playwright.
- AI component detection, content modeling, and mapping.
- Export formatting for Webflow.

## Out of Scope (for now)
- Direct Webflow API integration.
- Batch processing and custom templates (roadmap).
